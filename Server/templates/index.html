CBSE Result Parser
*Disclaimer: This site is not related to any manner with CBSE. It's a private site with Source Code publicly available.
CBSE Result Parser website requires user to upload the text file provided by CBSE. The uploaded file is then converted into excel file and is downloaded for the user. There is no data being stored from the file, Therefore user privacy is mantained. Source code is also made public, corrections and reccomendations are welcome.
Below are some frequently asked questions answered

FAQs

CBSE provides the result of class 10 and class 12 to schools in a text file. Upload that text file to convert the file into a excel file.

Excel file consists of list of students, with Subject Name as Columns Headers and their marks as Column values. For class 12, students are seperated into their respective streams in different sheets. PI value for each subject is also provided in a seperate PI sheet.
